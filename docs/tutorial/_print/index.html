Loredux の入門的な解説

本チュートリアルでは，Loreduxのメソッド全体について，テクニカルでない説明が展開されます．チュートリアルを一通り読むことで，メソッドを効果的に使用できるようになることが目指されています．

1 - イントロダクション

同期の法則

セルフヘルプや幸福のための処方箋，または願望実現のテクニックの中には，（「引き寄せの法則」として知られる一般化のように）心と現実があるレベルで同期していることに言及するものが数多くあります．Loreduxのメソッドにおいても，「ある人の現実は常にその人の信念体系に適合している」と表現される一般化が真であると前提されています（同期の法則）．

同期の法則は常識的・科学的世界観と整合しないように見えますが，それが問題なわけではありません．策を弄して不整合を見かけ上のものにすることは可能です．本当の問題は，同期の法則を受容するだけでは，殆どのケースで実践的な役には立たないという点にあります． 同期の法則を説く教説の多くにおいて，望む状況に意識を向けること，そして，望まない状況に意識を向けないことが推奨されています．この方針自体は正しいのですが，大抵は一時的な気休めにしかならず，望む状況が実現することはありません．

なぜでしょうか．あなたは欲求不満や不幸を感じており，だからこそ同期の法則を利用して，あなたの現実を変えたいと思います．そして，教説に従い望む未来に意識を向けます．しかし，このプロセスの中にあなたの信念体系が本当に変化するようなポイントがあるでしょうか．同期の法則によれば，あなたが不満を持っている現在の状況は，あなたの現在の信念体系に適合しています．望む状況に意識を向けるだけではこの信念体系に有意味な変化は起こりません．既存の信念体系が逆方向に引き付けるため，あなたは望む状況が実現することを本心から信じることができません．あるいは，一時的にその気になったとしても，望まない状況への恐れと不安を消すことができないので，結局その効果は相殺されてしまいます．それゆえに，あなたの現実にも望む変化が起きないのです．

どうすればよいのか

あなたを現に不幸にした信念体系を抱えたままでは決して事態は好転しません．しかも，同期の法則を自覚的に利用できるようになるには，あなたが思っている以上に，信念体系を深いレベルで改変する必要があるのです．しかし，誰もその方法を教えてくれません．Loreduxはまさにこの方法を提供するために開発されました．それは信念体系を根本的に改変して，同期の法則の自覚的な利用へと最適化します．

なお，Loreduxは同期の法則それ自体を扱うものではなく，それを利用できるように信念体系を改変する方法を教えるものです．同期の法則は物理法則でも数学的法則でもなく，概念分析や理論構築の対象にはなりません．それは我々の言語で完全に記述できるものではないのです．ただし，次のことは指摘しておくに値します．あなたの現実と同期しているのは，あなたの信念体系の全体というよりは，むしろその部分，あなたの感情に関与しているような部分であるということです．メソッドが不快な感情を一次的なトリガーとしているのは，そのような事情によります．もっとも，信念体系の要素は互いに連関しているため，ある感情に関与している信念は思ったよりずっと広範囲に分布しています．

また，最適化は度合いの問題です．メソッドによって信念体系の最適化が進行すればするほど，同期の法則を自覚的に利用できる度合いは高くなっていきます．あなたは，望む状況を想像してその実現を期待することを純粋に楽しめるようになるでしょう．不安と恐怖をもたらし逆方向に引き付ける信念が弱まっていくからです．この段階に至って初めて，同期の法則を説く様々な教説が言うような変化があなたに起きてくるのです．

2 - あなたが信じていること

問題の信念群

同期の法則を利用して望む状況を実現するためには，その状況をリアルに期待して，しかも良い気分でいる必要があります．しかし，往々にして既存の信念群が，期待や快い感情の持続を，それゆえに望む状況の実現を妨げてしまいます．問題の信念群とは，結局のところ，望む状況と両立しない状況が実現することを記述する信念群のことです．メソッドの手順に入る前に，このような信念群の特徴を整理しておきましょう．

まず言葉の用法を確定しておきます．人 $x$ が時間 $t$ において肯定する文の集合を「 $x$ の $t$ における信念体系」と言うことにします．ここでの「肯定する」は，もし尋ねられればその文に同意するであろう，という意味です．本心から同意しない可能性等は，差し当たり度外視します．信念体系は文の集合であり，そのメンバーである文が個々の信念です．この集合を「体系」と呼ぶ理由は，そのメンバー同士が論理的その他の連関によって結びついているからです．また，信念であれ何であれ，ある文がある状況の実現を記述していると言う場合，この「状況」には何らかの条件が成立する蓋然性も含まれます．したがって，条件 $\alpha$ が成立するレベル $i$ の蓋然性と，条件 $\beta$ が成立するレベル $j$ の蓋然性が事実上両立しない，などと言うことも有意味であるとします．なお，蓋然性の正確な定義は foundation,D73 にあります．それによると，蓋然性レベルが $1$ であるとき， $\alpha$ の蓋然性は $\alpha$ と基本的に同一になります．

挫折の図式

条件 $\alpha$ の成立を望んでいる場合，ある期間 $t$ の間， $\alpha$ が成立する高度の蓋然性を信じていれば（そして良い気分でいるならば），同期の法則によって $\alpha$ は実現するでしょう．しかし， $t$ のどこかで，

  1. 望まない条件 $ \beta $ の（レベル $j$ の）蓋然性が成立すること
  2. 1が $\alpha$ の高度蓋然性（ある高度レベル $i$ の蓋然性）と両立しないこと

の両方を信じているならば，その時点では通常， $\alpha$ の高度蓋然性の信念を持つことはできません．その結果， $\alpha$ の実現は妨げられます．それだけでなく，もし1について心配する等不快な感情を抱いているならば，同期の法則によって，望まない $\beta$ が実現することがあります．不快な感情が強く継続的である程，現実はそれに同期しやすくなってしまいます．

これが人生の数多くの場面で起きている挫折の図式です．例えば，あなたは自分が大金を得た状況を思い描きます．しかし，それと同時かすぐ後に来月の支払を心配します．その結果，大金を得る未来の実現が妨げられるのです．この場合，あなたは結局，自分が大金を得るより，それとは両立しない貧困の継続の方を，ずっと見込みが高いものとして信じているのです．つまり，あなたは自分が大金を得ることなど本当は信じていません．それは結局，同期の法則自体を信じていないということにもなりますが，この世界についての記述（を肯定する信念）と同期の法則（を肯定する信念）を同一のレベルで扱わない方がよいでしょう．

メソッドが変えるもの

上記の挫折の図式を考慮すると，望む条件 $\alpha$ を実現するためには，関連し合う次の2つの課題をクリアする必要があることが分かります．課題の1つ目は， $\alpha$ の成立と両立しない $\beta$ の成立を防ぐために，同期の法則が働かないように， $\beta$ が成立することへの不安等の不快な感情を無害化することです．不快な感情は信念体系の評価的な部分，すなわち価値的・規範的判断に依存しているか，少なくともそれによって増幅されているケースが多々あります．メソッドにおける論理的還元のステップは，この価値的・規範的判断を改変することによって，不快な感情を無害化します．

課題の2つ目は，$\alpha$ の高度蓋然性の信念を妨げないように，挫折の図式1と2を肯定する信念を改変することです．特にその信念に事実上伴うであろう以下のような因果的判断に関する改変が重要になります．すなわち，

  • $\beta^{\prime} = \beta $ のレベル $j$ の蓋然性，そして， $\alpha^{\prime}=\alpha$ の高度蓋然性，とします．すると，挫折の図式2が意味することは， $\beta^{\prime}$ と $\alpha^{\prime}$ の一方の実現が他方を因果的に阻止するか，または，単に $\beta^{\prime}$ と $\alpha^{\prime}$ が共通のメンバーを持たない，ということです．後者の典型例は， $j=1$ で， $\beta^{\prime}=\beta$ が単に $\alpha^{\prime}$ が成立しないという条件であるケースです．
  • 次に，挫折の図式1の $\beta^{\prime}$ を肯定する信念は，通常，他の条件 $\gamma$ の成立を肯定する信念と， $\gamma$ の成立によって $\beta^{\prime}$ の成立が因果的に決定されるとの信念から，推論によって形成されると考えられます．上記の例で言うと，①現在の収支の状態と，②①が来月の支払いに困る状況を因果的に決定していることを信じている場合，そこから推論過程によって，来月の支払いに困るだろうとの信念が形成され得るということです．

メソッドの存在論的還元のステップは，この部分の改変に関係しています．上記のような因果的判断は，全く合理的で常識的な判断かもしれません．つまり，$\alpha$ の高度蓋然性の信念を保持したいからといって，単に偽であるとして片付けることはできない類のものであり得るのです．そこで，メソッドにおいては，事実的・因果的判断をある構造に相対化する方向で信念体系を調整します．すなわち，構造 $x$ においては， $\beta^{\prime}$ が成立して， $\alpha^{\prime}$ が成立しない．他方，構造 $y(\neq x)$ においては， $\beta^{\prime}$ が成立せず， $\alpha^{\prime}$ が成立する，というように．そして，現実性はこのような構造が持つ属性であり，同期の法則による現実化とは，例えば構造 $x$ から構造 $y$ へとその属性を切り替えることだということも示唆されるでしょう．
