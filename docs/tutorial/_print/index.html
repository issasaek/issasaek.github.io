1 - イントロダクション

同期の法則

セルフヘルプや幸福のための処方箋，または願望実現のテクニックの中には，（「引き寄せの法則」として知られる一般化のように）心と現実があるレベルで同期していることに言及するものが数多くあります．Loreduxのメソッドにおいても，「ある人の現実は常にその人の信念体系に適合している」と表現される一般化が真であると前提されています（同期の法則）．ただし，現実と同期しているのは，信念体系の全体ではなく，感情の生起に関与しているようなその部分です．

同期の法則は常識的・科学的世界観と整合しないように見えますが，それが問題なわけではありません．策を弄して不整合を見かけ上のものにすることは可能です．本当の問題は，同期の法則を受容するだけでは，殆どのケースで実践的な役には立たないという点にあります． 同期の法則を説く教説の多くにおいて，望む状況に意識を向けること，そして，望まない状況に意識を向けないことが推奨されています．この方針自体は正しいのですが，大抵は一時的な気休めにしかならず，望む状況が実現することはありません．

なぜでしょうか．あなたは欲求不満や不幸を感じており，だからこそ同期の法則を利用して，あなたの現実を変えたいと思います．そして，教説に従い望む未来に意識を向けます．しかし，このプロセスの中にあなたの信念体系が本当に変化するようなポイントがあるでしょうか．同期の法則によれば，あなたが不満を持っている現在の状況は，あなたの現在の信念体系に適合しています．望む状況に意識を向けるだけではこの信念体系に有意味な変化は起こりません．既存の信念体系が逆方向に引き付けるため，あなたは望む状況が実現することを本心から信じることができません．あるいは，一時的にその気になったとしても，望まない状況への恐れと不安を消すことができないので，結局その効果は相殺されてしまいます．それゆえに，あなたの現実にも望む変化が起きないのです．

挫折の図式

同期の法則の利用に失敗する仕方について，さらに詳しく見てみましょう．例えば，あなたは自分が大金を得た状況を思い描きます．しかし，それと同時かすぐ後に来月の支払を心配します．その結果，大金を得る未来の実現が妨げられるのです．この場合，あなたは結局，自分が大金を得るより，それとは両立しない貧困の継続の方を，ずっと見込みが高いものとして信じているのです．つまり，あなたは自分が大金を得ることなど本当は信じていません．それだけでなく，来月の支払を心配することにより，同期の法則がむしろ貧困の継続の方を現実化してしまいます．その心配が強く持続性が高い程，現実はそれに同期しやすくなります．

これは人生の数多くの場面で起きている挫折の図式とでも言うべきものの例です．すなわち，同期の法則を利用して望む状況を実現するためには，その状況をリアルに期待して，しかも良い気分でいる必要があります．それにも関わらず，あなたは往々にして，望む状況と両立しない状況が実現することを心配してしまいます．それはつまり，望む状況が実現するよりも望まない状況が実現する見込みの方が高い，と信じているということです．

同期の法則を自覚的に利用するには，この挫折の図式を乗り越える必要があります．しかし，誰もその方法を教えてくれません．Loreduxはまさにこの方法を提供するために開発されました．それは信念体系を根本的に改変して，同期の法則の自覚的な利用へと最適化します．ただし，Loreduxは同期の法則それ自体を扱うものではありません．同期の法則は物理法則でも数学的法則でもなく，概念分析や理論構築の対象にはなりません．それは我々の言語で完全に記述できるものではないのです．

メソッドが変えるもの

挫折の図式を考慮すると，同期の法則を利用するためには，次の2つの課題をクリアする必要があることが分かります．

  1. 望む状況が実現することを妨げないように，それと両立しない状況（望まない状況）が同期の法則によって現実化するのを防ぐこと
  2. 望む状況が実現するとの信念を妨げないように，それと両立しない状況（望まない状況）が実現するとの信念を改変すること

既に述べた例で言えば，あなたが同期の法則によって大金を得るためには，それと両立しない貧困の継続が現実化することを防ぐ必要があります．また，大金を得ることを信じられるように，貧困が継続するとの信念をどうにかして阻止するか改変する必要があります．

まず，Loredux のメソッドは，望まない状況に対して同期の法則が働かないように，望まない状況への恐れと不安を無害化することによって，課題1を解決します．不快な感情は信念体系の評価的な部分，すなわち価値的・規範的判断に依存しているか，少なくともそれによって増幅されているケースが多々あります．メソッドは，この価値的・規範的判断を改変することによって，不快な感情を無害化します．なお，信念体系の要素は互いに連関しているため，ある感情に関与している信念は思ったよりずっと広範囲に分布しています．

次に，課題2については，望む状況と両立しない状況が実現するとの信念は，通常，現在の状況がそれを因果的に決定しているとの信念に基づいています．再び同じ例を用いれば，あなたは，現在の収支等の状況が貧困の継続（あるいは来月の支払いに困ること）を因果的に決定している，と信じており，そこから推論過程によって，貧困が継続するとの信念が形成されるということです．このような事実的・因果的判断は全く常識的で，異論の余地がないものであり得ます．つまり，これと両立しない大金を得ることを信じることは心理的に不可能であり得るのです． そこで，メソッドにおいては，事実的・因果的判断をある構造に相対化する方向で信念体系を調整します．すなわち，構造 $x$ においては，望まない状況が成立して，望む状況が成立しない．他方，別の構造 $y$ においては，望まない状況が成立せず，望む状況が成立する，というように．このような方向で，大金を得ることと貧困が継続すること，及びそれらの信念が両立可能になるのです．そして，現実性はこのような構造が持つ属性であり，したがって，同期の法則による現実化とは，例えば構造 $x$ から構造 $y$ へとその属性を切り替えることに他ならないことになります．

最後に，最適化は度合いの問題です．メソッドによって信念体系の最適化が進行すればするほど，同期の法則を自覚的に利用できる度合いは高くなっていきます．あなたは，望む状況を想像してその実現を期待することを純粋に楽しめるようになるでしょう．不安と恐怖をもたらし逆方向に引き付ける信念が弱まっていくからです．この段階に至って初めて，同期の法則を説く様々な教説が言うような変化があなたに起きてくるのです．
